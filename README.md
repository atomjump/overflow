# overflow

Acts like an 'overflow' on a sink: when there are too many messages in a forum, the old messages are trimmed.

# Requirements

AtomJump Messaging Server >= 0.8.0


# Installation


```
sudo php install.php
```

Include "overflow" in your config.json plugins.
